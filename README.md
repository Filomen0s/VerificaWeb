# VerificaWeb

🔎 Uma extensão de navegador criada para ajudar usuários a identificar possíveis fake news e verificar a confiabilidade de conteúdos encontrados na internet.

Sobre o Projeto

O VerificaWeb é uma iniciativa voltada ao combate à desinformação digital. O projeto propõe o desenvolvimento de uma extensão de navegador capaz de analisar conteúdos de páginas web e fornecer uma avaliação sobre a confiabilidade das informações apresentadas.

Ao acessar um site, a extensão realizará uma análise do conteúdo e classificará a informação em uma das seguintes categorias:

✅ Confiável — Conteúdo respaldado por fontes confiáveis e evidências verificáveis.
⚠️ Duvidoso — Conteúdo com informações insuficientes, inconsistentes ou sem comprovação clara.
❌ Falso — Conteúdo com indícios de desinformação ou contradito por fontes verificadas.

O objetivo é auxiliar usuários na tomada de decisões mais informadas e incentivar uma navegação mais segura e consciente.
